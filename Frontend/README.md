
  # English Learning App UI

  This is a code bundle for English Learning App UI. The original project is available at https://www.figma.com/design/B2xh6Lq3BcHCmQCPyKI9bO/English-Learning-App-UI.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  我會根據目前的項目結構，整理一份適合你直接上手的開發導覽，重點放在入口、主要模塊和後續擴展方向。

## 開發導覽

這是一個以 React + Vite 為核心的英語片語學習前端原型，現在的重點是「界面、互動與學習流程」，還沒有接真正的後端或資料庫。

---

## 1. 先怎麼啟動

在 Frontend 目錄下執行：

```bash
npm install
npm run dev
```

開發服務啟動後，通常會在本地 Vite 服務上看到預覽地址。

---

## 2. 你現在最該先看哪幾個文件

### 入口與主流程
- main.tsx
  - 應用啟動入口

- App.tsx
  - 這是整個 App 的核心文件
  - 包含 Home、Library、Settings 三個主畫面
  - 也包含所有 mock data、UI 組件與互動邏輯

### UI 元件
- ui
  - 這裡是可重用的 UI 基礎元件，例如按鈕、卡片、輸入框等

### 樣式與主題
- styles
  - 全域樣式與主題設定

### 專案設定
- package.json
  - 依賴與腳本
- vite.config.ts
  - Vite 設定

## 3. 目前專案的架構理解

### 目前是什麼型態
這個項目目前更像是：
-「產品原型」
-「前端 UI demo」
-「可互動的學習流程展示」

它不是一個完整的 SaaS 產品，因為目前：
- 資料是寫死在前端的
- 沒有後端 API
- 沒有使用者帳號系統
- 沒有真實持久化儲存

---

## 4. 主要功能分層

### Home
負責學習流程：
- 抽取一個 chunk
- 顯示翻譯與例句
- 做填空練習
- 進行下一題

### Library
負責內容管理：
- 顯示所有片語
- 搜尋
- 分類篩選
- 標記需要複習或已掌握

### Settings
負責偏好設定：
- 每日目標
- 提醒開關
- 音效與震動偏好

---

## 5. 開發時最常修改的地方

### 如果你想改內容
直接修改 App.tsx 裡的資料陣列，例如：
- 學習片語資料
- 分類
- 每週進度資料

### 如果你想改畫面
主要也是在 App.tsx 裡改：
- 卡片樣式
- 按鈕樣式
- 各頁佈局

### 如果你想新增通用元件
可以在 ui 裡新增，之後再由主頁引用。

---

## 6. 接下來最適合的擴展方向

### 第一優先
1. 把 mock data 改成真實資料來源
2. 加入本地儲存，讓進度不會刷新後消失
3. 加入路由，讓頁面更像正式 App

### 第二優先
4. 把樣式從內聯樣式整理成更可維護的方式
5. 加入 API / 後端串接
6. 增加使用者帳號與收藏/複習系統

---

## 7. 開發建議

- 先不要一開始就改太大架構，先把「一個功能跑通」最有效
- 這個專案目前最適合做「UI 與互動優化」
- 如果你要做產品化，下一步最值得做的是：
  - 資料串接
  - 狀態持久化
  - 使用者流程完整化
